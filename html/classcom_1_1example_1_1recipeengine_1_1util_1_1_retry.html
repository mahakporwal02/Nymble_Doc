static

Calls consumer with the argument argumentForConsumer, if the call completes without an exception, the method returns. If the call doesn't completes and throws an error, then the error can be of type expectedException or any other type. If error of expectedException type, then the method call is retried maximum of maxAttemptCount. If the error is not of expectedException type, then the method bubbles up the error.
