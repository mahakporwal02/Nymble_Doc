Detailed Description

This class extends BaseCookDecorator and applies OneTime Decoration to a com.example.recipeengine.instruction.cook.Cook Object. It overrides the methods getDefaultWattage() to return the modified defaultWattage, getTemperature() to return the modified temperature and getTimeToCook() to return modified timeToCook.
