Detailed Description

This class is a blueprint of a Cook Object. It implements BaseCook and extends BaseInstruction and it overrides the methods of ActionParams CaptureParams, StirrerParams,InferParams and adapt() which calls theCookAdaptor#adapt() method and returns the adaptedSaute object.
