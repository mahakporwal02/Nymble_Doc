Detailed Description

This class is the implementation of InstructionAdaptor, which overrides applyOneTimeDecoration() which is responsible for applying decoration on Dispense objects which have attemptNum as 0. and applyCompensatoryDecoration() which is responsible for applying decoration

  • on Dispense objects which have attemptNum greater than 0.
