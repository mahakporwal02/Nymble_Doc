Detailed Description

This class is responsible for running the CapturableTask in parallel using startCapture() also it stores the results in CapturedTaskData using consolidateTaskResult(List) .

List<CapturedTaskData> com.example.recipeengine.instruction.blocks.capture.BaseCaptureBlock.getCapturedData ( ) throws Exception

The client uses this method to get the List of CapturedTaskData.

  1. It first checks if the capturedBlockData member is null or not
  2. Then it calls the startCapture() method, which returns List of CapturedTaskData.
  3. The same is returned to parent method.
