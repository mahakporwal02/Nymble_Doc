◆ run()

void com.example.recipeengine.instruction.task.capture.vision.threads.ThermalCameraThread.run ( )

This function contains the logic of capturing Thermal Images.

  1. First we run a for loop for how many samples we are trying to collect, it is present in SensorParams object.
  2. Then we create an object Sensor and set the sensor to Thermal Camera.
  3. Then we call the RequestHandler#handleRequest(Request) method to execute the given request and return the SensorResponse.
  4. The sensor response returns a Map (String,List Double object and we store it by converting it into a Bitmap image using BitmapUtility#convertToBitmap(List) method.
  5. The current thread is put to sleep for a specific amount of time mentioned in the sensorParams.
