Detailed Description

This class is a blueprint of a Dispense Object. It implements BaseDispense and extends BaseInstruction. The methods this class overrides are getQuantityInGrams() which returns quantity of the Dispense Instruction getIngredients() which returns the list of ingredients. adapt() which calls the DispenseAdaptor#adapt() function and returns the adaptedDispense.
