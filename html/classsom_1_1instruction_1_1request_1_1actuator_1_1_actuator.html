static HeatPowerLevel som.instruction.request.actuator.Actuator.createHeatLevel ( Integer  heatLevel)
static

Accepts an integer, which represents a power level and converts it into one of the enums from HeatPowerLevel. Quantizes the integer value of power level into the eight power levels starting from 0 to 1400 at a gap of 200 watts(0,200,400...1200,1400),corresponding to HeatPowerLevel. First, this method finds out the closest integral value from the set S = {0,200,400,600,800,1000,1200,1400}, using the quantization rule : {find the closest integer for the given heat level from the set S, if the given heat level lies midway from two power levels then pick up the smaller power level} then it finds out the enum using the following rule : POWER_LEVEL_X is the enum for X*200 quantized integral value. For example, an integer value of 200 should result in the enum HeatPowerLevel#POWER_LEVEL_ONE an integer value of 490 should result in the enum HeatPowerLevel#POWER_LEVEL_TWO
