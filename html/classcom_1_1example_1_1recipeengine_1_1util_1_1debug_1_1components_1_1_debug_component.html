void com.example.recipeengine.util.debug.components.DebugComponent< B extends ViewDataBinding >.close ( )

Filter the feedback messages with a condition.

Returning false will give no response (ie) the method onResponse(String) will never be called.

Returning true will allow all messages (ie) the method onResponse(String) will be called for all MCU responses.
